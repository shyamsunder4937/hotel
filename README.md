# Hotel Management System

A full-stack Hotel Management application with a frontend and backend. 

## Project Structure

- **hotel-frontend**: The user interface of the application, built with modern web technologies. Contains booking, admin dashboard, and user-facing pages.
- **hotel-backend**: The API server and database controllers, handling reservations, room availability, payments, and admin logic.

## Prerequisites

- Node.js installed
- Database credentials configured (add a `.env` file to your backend directory if necessary)

## Setup Instructions

### 1. Backend Setup
Navigate to the backend directory, install dependencies, and start the server:
```bash
cd hotel-backend
npm install
npm run dev
```

### 2. Frontend Setup
Navigate to the frontend directory, install dependencies, and start the development server:
```bash
cd hotel-frontend
npm install
npm run dev
```

## Features

- **User Booking**: Browse rooms and book stays.
- **Admin Dashboard**: Manage reservations, rooms, and view overall statistics.
- **Responsive Design**: Interfaces built to work on mobile, tablet, and desktop devices.
